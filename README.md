# ⚡ Smart City Energy Insights Dashboard

> A comprehensive **Power BI Dashboard** developed to analyze smart city energy consumption, revenue, tariff performance, outages, and meter insights. The project enables interactive monitoring of energy usage and operational KPIs through multiple report pages.

---

# 📌 Project Overview

This Power BI report focuses on analyzing smart city energy data using interactive dashboards. It provides insights into:

- ⚡ Energy Consumption
- 💰 Revenue & Tariff Performance
- 🚨 Outage Monitoring
- 🔌 Smart Meter Analysis

The dashboard is designed for quick decision-making using filters, KPI cards, and interactive visualizations.
